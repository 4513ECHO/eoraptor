## How to Run

You have to set `POSTGRES_URL` envrironment variable. In local, you can set it
through `.env` file. In Deno Deploy, you can set it from configuration.

```sh
POSTGRES_URL="postgres://{user}:{password}@{domain}:{post}/{database}"
```
